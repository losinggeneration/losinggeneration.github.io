CrabEmu for the Didj

2011-04-28 :: losinggeneration
#ARM  #Buildroot  #C  #Didj  #DJHI  #uClibc 
Well, I’m releasing what I’ll call version 0.1.0 of CrabEmu for the Didj. It includes a custom built rootfs with no questionable code from the LF distribution (for instance, their scripts for checking various things like USB connections.) This initial release requires a DJHI or other third party card that is SD capable. The SD must also have two partitions: Fat32 & Ext 3. Because of the later, you’ll also need an OS that’s able to read/write to Ext 3 file systems.

Didj OpenGL ES

#Didj  #OpenGL ES 
Well, I may be the first to get even basic OpenGL ES primitives to render. I can now make my claim that NeHe lesson02 works for the Didj. Some others have things such as OpenGL version strings, but I believe this is the first actual rendering for Didj homebrew. I’ve gone a slightly different route then what others have done. I’ve used Brio directly to satisfy some of the OpenGL ES dependencies.

DJHI

2010-04-21 :: losinggeneration
#Buildcross  #Didj  #DJHI  #Linux 
Well, I now have my Didj, a Serial->USB breakoutboard, and a DJHI v2.0 prototype board. With these I’m able to get serial access to the Didj. I’ve flashed a new version of the Lightning Boot bootloader which jburks has been hard at work on. It adds a boot menu which is very useful for booting new kernels off of an SD card. jburks also has a preliminary kernel driver to access the SD card from within Linux.
